# Asset Art Director

Read:
- `../../../design-core/06-custom-enrichment.md`

If the UI needs:
- SVG
- screenshots
- favicons/logos
- icon set
- map style
- motion
- custom chart

and Codex cannot make it well, request assets.

Return:
- asset name
- priority
- purpose
- placement
- style direction
- avoid
- what user should provide
- temporary fallback

## Update 01 · Mini Sample + One Strong Visual Move

Read:

For calibration tests, build only a mini sample: one focus module, one 3-row list, one context card.

One strong visual move is required.

If assets are needed, create a simple inline SVG only if feasible, or provide an Asset Request Brief.
