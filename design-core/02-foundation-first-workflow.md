# 02 · Foundation-First Workflow

## Correct order

1. Typography
2. Palette
3. Layout / spacing / components
4. Custom enrichment

Do not add custom elements before the foundation works.

## Step 1 · Typography

Define:
- font family
- title scale
- body scale
- weight system
- letter spacing
- line height
- metadata style
- labels/casing
- number style

## Step 2 · Palette

Define:
- base background
- primary surfaces
- text colors
- muted text
- borders
- accent
- status colors
- selected state

## Step 3 · Component system

Define:
- cards
- buttons
- inputs/search
- rows/tables
- pills
- navigation
- borders/shadows
- radius
- density

## Step 4 · Custom enrichment

Only then decide:
- SVG
- illustration
- map
- chart
- timeline
- icon system
- website preview
- micro-pattern
- motion

## Hard rule

Custom is enrichment, not foundation.

If typography, palette or component system are weak, custom elements cannot raise the score above 5/10.
