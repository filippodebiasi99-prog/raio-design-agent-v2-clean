# Failure · Generic Mini Sample With Placeholders

## Symptoms

The agent creates:
- one focus card
- one lead list
- one side card

but the result still looks generic because:
- typography is only slightly improved
- palette is basic
- focus module is just a normal card
- list rows are standard
- identity markers are random initials
- website previews are fake skeletons
- no real visual move exists
- no asset request is made

## Why it fails

The task was mini, but the output was not designed.

Mini samples still require one strong visual move.

## Fix

Before coding:
- choose one primary visual move
- decide whether assets are needed
- if visual move depends on assets, request them or create a simple inline SVG
- do not deliver placeholder-heavy output as final

## Score

A generic mini sample with weak placeholders cannot score above 5/10.
