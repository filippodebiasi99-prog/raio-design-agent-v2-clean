# Reference 02 · Fluid Sidebar + SVG

## Why it feels beautiful
Not because the sidebar is black, but because navigation and illustration are treated as designed/brand-specific elements.

## Typography notes
Small, delicate, low-bold text. Avoid huge hero type.

## Palette notes
High contrast can work when controlled. Do not make every UI dark.

## Layout / spacing notes
Some parts are too empty, so do not copy full layout blindly.

## Component notes
Sidebar active state is memorable. Hero card with SVG is more successful than generic metric cards.

## Custom enrichment notes
SVG illustration adds brand identity.

## What not to copy literally
Do not require black sidebar. Do not copy follower/metric sections.

## Transferable rule
Even basic UI parts can become custom-designed brand elements.

## RaiOS relevance
Use for navigation/illustration inspiration only if foundation is already strong.
