# 12 · One Strong Visual Move

A RaiO mini sample must have one clear visual move.

Not ten. One.

## Acceptable visual moves

Choose one:

### A · Branded focus field
A soft, custom field inside the focus module with refined typography, subtle visual detail and compact CTA.

### B · Premium CRM row treatment
The 3-row lead list becomes the visual hero through typography, spacing, identity markers, refined selected state and status system.

### C · Custom operational path
A small audit → demo → outreach path becomes the branded detail, but it must be elegant and compact.

### D · Branded SVG / illustration
A small custom inline SVG or illustration enriches the focus module or side card.

### E · Website preview as visual anchor
A more credible mini website preview becomes the main custom detail.

## What does not count

These do NOT count as strong visual moves:
- colored initial boxes
- generic pills
- random score number
- default icon rail
- fake browser skeleton
- subtle background gradient only
- adding a CTA button
- adding a right card
- adding a table row

## Required statement

Before coding, state:

```md
Primary visual move:
Why it supports RaiOS:
Why it is not just decoration:
How it is grounded in the references:
```

## If no visual move is present

Overall score cannot exceed 5/10.

## If visual move is weak

If the selected visual move is generic, crude, or barely visible, revise before delivery.
