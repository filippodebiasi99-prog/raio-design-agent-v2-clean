# Reference 07 · Soft Purple Learning Dashboard

## Why it feels beautiful
Not because it is purple, but because the hero module has calibrated typography, color field, subtle background detail and a compact CTA.

## Typography notes
Headline feels pleasant and controlled. Text is not overly bold.

## Palette notes
Purple works because it is clean and intentional. Do not copy color literally.

## Layout / spacing notes
Topbar/search spacing is well structured. Content modules are friendly.

## Component notes
Hero card is strong. Category modules may be too large. Chart is not custom enough.

## Custom enrichment notes
Subtle stars/details enrich the card. Avatars add warmth.

## What not to copy literally
Do not copy purple just because it is purple.

## Transferable rule
A home/focus module can create arrival and personality if foundation is well calibrated.

## RaiOS relevance
Use for dashboard home richness and focus module quality.
