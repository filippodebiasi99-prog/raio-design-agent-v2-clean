# Skill Additions

## reference-selector
Add:
- choose one primary reference stack for the mini sample
- do not mix many references

## foundation-director
Add:
- define the mini sample foundation before code
- keep typography/palette/components focused

## ui-builder
Add:
- build only the mini sample scope
- no full dashboard shell
- one strong visual move required

## asset-art-director
Add:
- run asset decision before code
- create small inline SVG only when feasible
- otherwise request asset

## visual-reviewer
Add:
- score visual move strength
- score placeholder quality
- if no strong visual move, overall max 5/10
