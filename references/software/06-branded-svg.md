# Reference 06 · Branded SVG

## Why it feels beautiful
Mostly because the large custom illustration gives identity. The UI around it is not all good.

## Typography notes
Right panel has calm basic text. Left stats are not liked.

## Palette notes
Orange/black is contextual, not global.

## Layout / spacing notes
Right panel is basic but acceptable. Left is weak.

## Component notes
Do not copy stat cards/sidebar/calendar.

## Custom enrichment notes
SVG is a brand multiplier when good.

## What not to copy literally
Do not copy full UI.

## Transferable rule
A branded illustration can enrich a strong interface but cannot save a weak foundation.

## RaiOS relevance
Request SVG assets for home/focus/empty states when useful.
